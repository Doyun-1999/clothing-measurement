# 프로젝트명
> 의류 실측을 통한 중고 거래 앱



스마트폰 카메라로 찍은 의류 이미지를 통해서 의류의 실제 수치를 측정하여 중고 거래 어플리케이션에 활용한다.

## 핵심 기능
* 접근성을 높이기 위해 휴대폰 카메라를 활용하여 측정할 수 있도록 제작한다.
* 의류 이미지의 외곽선을 추출하여 각 부위별(어깨너비, 가슴둘레, 총장, 소매길이) cm 단위로 환산하여 수학적인 데이터를 표기한다. 
*  커뮤니티를 형성하여 모바일 중고 거래를 활성화한다.
